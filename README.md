# استراتيجية الفراكتل

## وصف
هذه الصفحة تعرض استراتيجية الفراكتل ومفاهيمها الأساسية. تهدف الاستراتيجية إلى التنبؤ بالسلوك الرائد القادم والمجهول من خلال تحليل السلوك السابق. تعتمد على فكرة أن التاريخ يعيد نفسه من خلال سلوكيات معينة.

## المفاهيم الأساسية
- **الفراكتل**: علم رياضي يتعامل مع قياسات غير فورية.
- **التناسخ والتكرار**: تكرار الحركة السلوكية بين الإرسال.
- **التصغير والتكبير**: كل حركة سعرية تستقر إلى تصغير ومكبر.
- **الناسخ والمنسوخ**: السلوك الذي يتكرر ليظهر تشابهًا.
- **السويت زون**: أفضل منطقة للشراء أو البيع.
